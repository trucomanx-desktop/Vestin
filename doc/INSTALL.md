# vestin

Program to calculate compound interest


## Install from PYPI

The homepage in pipy is https://pypi.org/project/vestin/

```bash
pip install --upgrade vestin
```

Using:

```bash
vestin
```

## Install from source
Installing `vestin` program

```bash
git clone https://github.com/trucomanx-desktop/Vestin.git
cd Vestin
pip install -r requirements.txt
cd src
python -m build
pip install dist/vestin-*.tar.gz
```
Using:

```bash
vestin
```

## Uninstall

```bash
pip uninstall vestin
```
